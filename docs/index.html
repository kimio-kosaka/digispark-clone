This is digispark(clone)'s developenvironment package for Arduino IDE.

Abailable

  • Windows
  • mac OS
  • PC Linux
  • Raspberry Pi(Raspbian)
  • able to make clones by "burn bootloader" on Arduino IDE.
below is additional board manager URL for Arduino IDE.
https://kimio-kosaka.github.io/digispark-clone/package_digispark-clone_index.json

On Raspbian

Set the USB device access rule by your hand

touch /etc/udev/rules.d/49-micronucleus.rules

add rule-codes to 49-micronucleus.rules file.

rule-codes

SUBSYSTEMS=="usb", ATTRS{idVendor}=="16d0", ATTRS{idProduct}=="0753", MODE:="0666"
KERNEL=="ttyACM*", ATTRS{idVendor}=="16d0", ATTRS{idProduct}=="0753", MODE:="0666", ENV{ID_MM_DEVICE_IGNORE}="1"
